PRIORITY ORDER FOR INTERVIEW PREP
================================================================================

  Study in this order — highest interview frequency first:

  1.  Dependency Injection — DI lifetimes are asked in almost every .NET interview
  2.  async / await — heavily asked for .NET and React roles
  3.  JWT Authentication — nearly universal in web API interviews
  4.  REST conventions + HTTP status codes — basic but always asked
  5.  Repository Pattern + Unit of Work — very common in .NET API interviews
  6.  SQL JOINs and Indexes — asked in most full-stack interviews
  7.  Custom Hooks (React) — commonly asked in React interviews
  8.  React Router — expected knowledge for any React role
  9.  Redux Toolkit — expected if the job mentions state management
  10. SOLID in depth with code examples — architecture interviews always ask this
  11. EF Core advanced — N+1, AsNoTracking, Include — asked in .NET interviews
  12. Docker basics — increasingly asked even in non-DevOps roles
  13. Clean Architecture — asked in senior/mid level .NET interviews
  14. Message brokers (RabbitMQ/Kafka) — asked in microservices roles
